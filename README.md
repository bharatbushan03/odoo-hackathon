# Hackathon Starter Kit

## Quick start
```bash
bash scripts/setup.sh
```
Then run backend and frontend in separate terminals (see docs/SETUP.md).

## Structure
- `backend/` — Express API
- `frontend/` — React (Vite) app
- `docs/` — API contract, setup, architecture notes
- `scripts/` — automation

## Before the sprint
1. Run setup.sh successfully on every laptop
2. Fill in docs/API_CONTRACT.md the moment the problem statement drops
3. Assign Integration Owner in docs/SETUP.m
